# TP4 - Cuerpo Consciente
Cuerpo Conciente Pilates es una red de profesionales dedicada a la enseñanza y difusión del pilates desde una perspectiva consciente, técnica y terapéutica. A partir del análisis de su sitio web original, detectamos problemas de navegación, organización de contenidos, jerarquización de la información e identidad visual, lo que afectaba la experiencia de usuario.
Nuestra propuesta consistió en rediseñar la web para lograr una estructura más clara, una identidad visual coherente y una navegación más intuitiva. Se reorganizaron los contenidos, se incorporó una nueva paleta cromática, tipografías más actuales y recursos visuales que aportan dinamismo sin sobrecargar la interfaz.
El proyecto fue diseñado en Figma y desarrollado con HTML, CSS y JavaScript, incorporando mejoras como animaciones, videos, sliders y una adaptación responsive para distintos dispositivos. El trabajo se realizó de manera colaborativa mediante GitHub, permitiendo transformar el prototipo en una web funcional, accesible y alineada con los objetivos de comunicación de Cuerpo Conciente Pilates.
